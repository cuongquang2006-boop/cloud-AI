# Cloud AI Server (Render)

### Notes
- The first response may take about 15 seconds.
- After that, responses will be faster.
- If there is no activity for 15 minutes, the server will go to sleep.

### Reason
- This server is deployed on Render free tier, so it may take time to start when it is inactive.

### Tips
- Send a request before using to wake up the server.
- Try to avoid long idle time if you need fast responses.
